Text Embedding REST API

The Text Embedding REST API allows the user to retrieve the text embeddings of a given text.

Available methods
GET /embeddings/create Create the text embedding
POST /embeddings/create Create the text embedding (similar to the GET request)
Create the text embedding
GET /embeddings/create

Route information

Query string parameters
text Type: String Title: Query Text Optional: false The text for which we wish to retrieve the text embedding.
language Type: String Title: Text Language Optional: true The language of the provided text.
Response body attributes
language_model Type: String Title: Language Model Read only: true The language model used to create the text embedding.
tokens Type: Array Title: Tokens Read only: true An array of objects, each representing a non-stopword token of the provided text.
tokens item properties
token Type: String Title: Token Name Read only: true The non-stopword token name.
count Type: Integer Title: Token Count Read only: true The frequency of the token in the original text.
embedding Type: Array Title: Text Embedding Read only: true An array of floats representing the text embedding.
text Type: Array Title: Source Text Read only: true The original source text.
